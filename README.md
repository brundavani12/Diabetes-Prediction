# Diabetes Prediction Using Machine Learning

## Overview

This project predicts whether a person is diabetic or not using Machine Learning techniques. The application is built using Flask and provides a simple web interface for users to enter health-related parameters and receive a prediction.

## Technologies Used

* Python
* Flask
* NumPy
* Scikit-Learn
* HTML
* Machine Learning

## Features

* User-friendly web interface
* Data preprocessing using imputer and scaler
* Diabetes prediction using a trained Random Forest model
* Real-time prediction results

## Project Structure

* app.py : Flask backend
* templates/index.html : Frontend interface
* rf_model.pkl : Trained Random Forest model
* scaler.pkl : Data scaler
* imputer.pkl : Missing value handler
